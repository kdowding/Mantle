# Code of Conduct

Mantle's issue tracker and discussions are small spaces attached to a personal
project. The bar is simple: **be decent to each other.**

## Expected

- Be respectful and constructive; critique code and ideas, not people.
- Assume good faith, especially across language and experience gaps.
- Keep discussions on-topic and harassment-free.

## Not tolerated

- Harassment, insults, or discriminatory remarks of any kind.
- Publishing others' private information.
- Sustained disruption, bad-faith argument, or spam.

## Enforcement

This is a single-maintainer project; enforcement is at the maintainer's
discretion and may include hiding comments, locking threads, or blocking
accounts. Report conduct issues to the maintainer via GitHub
([@kdowding](https://github.com/kdowding)); reports are handled privately.

Adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org/).
